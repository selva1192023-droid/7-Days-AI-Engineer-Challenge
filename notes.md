# Day 1 - AI Engineer Roadmap

## What is an AI Engineer?

An AI Engineer builds intelligent applications using Artificial Intelligence, Machine Learning, Deep Learning, and Large Language Models (LLMs).

## Skills Required

* Python
* Data Analysis
* Machine Learning
* Deep Learning
* Generative AI
* Prompt Engineering
* RAG
* AI Agents
* MLOps & Deployment
