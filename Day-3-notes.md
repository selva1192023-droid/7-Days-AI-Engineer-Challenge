# Day 3 - Machine Learning Fundamentals

## What is Machine Learning?

Machine Learning is a branch of Artificial Intelligence that enables computers to learn patterns from data and make predictions without being explicitly programmed.

## Types of Machine Learning

### Supervised Learning

Uses labeled data to train models.

Examples:

* House Price Prediction
* Student Score Prediction

### Unsupervised Learning

Uses unlabeled data to find patterns.

Examples:

* Customer Segmentation
* Clustering

## Key Concepts

* Features
* Labels
* Training Data
* Testing Data
* Accuracy
